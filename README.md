### Quick Customization Guide (Make It Yours!)
Just append these to the base URL `https://leetcard.jacoblin.cool/GautamKumar_code?`

- **Themes** ‚Äî try: `dracula`, `radical`, `tokyonight`, `onedark`, `gruvbox`, `synthwave`, `transparent`, `light`, `dark`...
- **Fonts** ‚Äî 1300+ Google Fonts! Examples: `Fira Code`, `JetBrains Mono`, `Roboto Mono`, `Source Code Pro`
- **Extensions** (add multiple with comma): `heatmap`, `contest`, `activity`
- **Extras**: `&border=0`, `&radius=20`, `&hide=hard` (if you want to flex easy/medium only üòè)

Example full beast mode:  
`?theme=dracula&font=JetBrains%20Mono&extension=heatmap,contest&radius=18&border=1`

This combination gives you **line graphs**, **heatmaps**, **progress bars**, **animated fills**, and **activity timelines** ‚Äî currently the most graphical & impressive setup possible without self-hosting custom SVGs.
